# Проект - виджет банковских операций клиента

## Цель проекта
Расширение возможностей банковских операций клиента 

## Инструкции по установке и использованию разработанных функций 
1. Скачайте проект
2. Установите окружение и зависимости согласно файлу pyproject.toml
3. Используйте функции по необходимости

## Примеры работы с ними
* для использования функции *get_mask_card_number* вызовите её, внося в качестве параметра номер карты клиента: ```get_mask_card_number(card_number)```
* для использования функции *get_mask_account* вызовите её, внося в качестве параметра номер счёта клиента: ```get_mask_account(account_number)```
* или сразу используйте функцию маскирования данных клиента *mask_account_card*, вводя в параметр инфр клиента ```mask_account_card(info_card)```
* модуль *`generators.py`*, пример использования функции `*card_number_generator*` - внесите начальное и конечное значения для генерации диапазона номеров и генератор выдаст номера карт из этого диапазона 


## Дополнительно
* Добавлен модуль `generators.py`.  Этот модуль содержит новые функции, реализующие генераторы для обработки данных. Также добавлен модуль ```test_generators.py``` для тестирования модуля `generators.py`. 
* Добавлен модуль `decorators.py`.  Этот модуль содержит новую функцию, реализующую декоратор для обработки исходной функции и логировании её в консоль или в отдельный файл. Также добавлен модуль ```test_decorators.py``` для тестирования модуля `decorators.py`.
